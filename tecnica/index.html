Capacidades técnicas

Técnica

Trabajo técnico orientado a proyectos artísticos, escénicos y de investigación. No es una lista de servicios comerciales: es un mapa de herramientas con las que construyo proyectos propios y colaboro con otros artistas.

Áreas de trabajo técnico

◈

Diseño audiovisual para escena

Diseño y operación de video, imagen y audio en contextos escénicos. Trabajo con proyección, mapping, procesamiento en tiempo real y sincronización con otros sistemas escénicos.

  • Proyección y mapping
  • Video en tiempo real
  • Sincronización con cues
◈

Sistemas de video en vivo

Procesamiento y composición de video en tiempo real. Integración con señales externas, sensores y sistemas de control. Construcción de pipelines de video para instalaciones y performances.

  • NDI / HDMI routing
  • Captura y procesamiento en tiempo real
◈

Instalación interactiva

Diseño e implementación de sistemas interactivos que responden a presencia, movimiento, sonido o datos ambientales. Desde el concepto hasta el montaje y operación.

  • Kinect / cámaras de profundidad
  • Sensores ambientales
  • Integración hardware-software
  • Protocolos OSC / MIDI / DMX
◈

Electrónica y sensores

Diseño y construcción de circuitos electrónicos a medida para proyectos artísticos. Microcontroladores, sensores de todo tipo, actuadores físicos y sistemas embebidos.

  • ESP32 / Arduino / Raspberry Pi
  • Sensores: luz, temperatura, ultrasonido, IMU
  • DAC / ADC / amplificación
  • Diseño de PCB básico
◈

Sonido experimental

Diseño sonoro experimental para instalaciones, escena y composición propia. Síntesis, procesamiento, espacialización y construcción de sistemas sonoros físicos.

  • Síntesis de audio
  • Procesamiento en tiempo real
  • Espacialización / multichannel
◈

Iluminación y DMX

Diseño y programación de sistemas de iluminación para escena e instalación. Control DMX512, integración con software de show-control y automatización de comportamientos.

  • DMX512 / Art-Net
  • PAR LED, Moving heads
  • Programación de shows y cues
◈

Software creativo

Desarrollo de herramientas de software orientadas a proyectos artísticos. Show-control, síntesis generativa, procesamiento de datos en tiempo real, interfaces para escena.

  • Rust / Tauri
  • Svelte / JavaScript
  • Python
◈

Operación técnica escénica

Operación técnica integral para proyectos escénicos y de performance. Coordinación de sistemas, programación de shows, pruebas y operación en vivo.

  • Show-control y cues
  • Integración de sistemas
  • Operación en directo
