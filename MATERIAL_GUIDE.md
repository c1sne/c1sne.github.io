# MATERIAL_GUIDE.md — Materiales pendientes por preparar

Esta guía lista todos los materiales que debes reunir para completar la web. Está organizada por sección.

---

## Statement / textos globales

- [ ] **Statement principal** (≈ 80–150 palabras): texto breve y fuerte que define tu práctica. Va en la página de inicio.
- [ ] **Bio corta** (≈ 80–100 palabras): para página de Bio y perfiles externos.
- [ ] **Bio larga** (≈ 300–500 palabras): para página de Bio completa.
- [ ] **Texto de investigación**: descripción de tus líneas de investigación en tus propias palabras.

Guardar en: `materiales/textos/`

---

## CV

- [ ] **CV en PDF** actualizado (formato A4, en español y/o inglés).
- [ ] **CV en texto plano o Markdown** para web (opcional pero útil para SEO).

Guardar en: `materiales/cv/` y copiar a `web/public/cv/`

Formato sugerido: `cv-andres-cisneros-2024.pdf`

---

## Imágenes globales

- [ ] **Foto de perfil / retrato profesional**: al menos 1 imagen, preferiblemente 2–3 opciones.
  - Formato: JPG o WebP
  - Tamaño mínimo: 800×800px
  - Guardar en: `materiales/imagenes/` y `web/public/images/`

- [ ] **Imagen Open Graph** (para redes sociales): 1200×630px, horizontal.
  - Puede ser logo, retrato, o imagen de proyecto destacado.

## Formatos recomendados

| Material | Formato | Tamaño máximo recomendado |
|---|---|---|
| Imágenes de portada | WebP o JPG | 1200×800px, < 300KB |
| Imágenes de galería | WebP o JPG | 1600px ancho, < 500KB |
| Foto de perfil | WebP o JPG | 800×800px, < 200KB |
| Open Graph | JPG | 1200×630px |
| Video embed | URL de Vimeo/YouTube | — |
| CV | PDF | < 5MB |
| Fichas técnicas | PDF | < 5MB |
